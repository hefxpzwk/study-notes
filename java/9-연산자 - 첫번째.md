# 9. 연산자 - 첫번째

## 연산자 기본 이론

- **연산 = 실행 = cpu가 명령을 수행**하는것
  (명령의 대부분이 연산에 대한 명령임)
- 연산자는 **cpu 연산과 직결**되는 문법

##### 연산식

- **연산자 + 피연산자**로 구성
- 필요한 **피연산자의 수**에 따라 **단항 연산자, 2항 연산자, 3항 연산자** 라고 부른다.
  - **'+'**는 이항연산자 항이 두개 필요해서
  - ` a + b`
